Decrypts a ciphertext under a private key. Computes t := c*B where c is the ciphertext and B is the private key, and returns true if the distance between t and the integer lattice is >d, otherwise returns false.

Encrypts a bit under a public key. If b=1, ciphertext is sampled uniformly from [0,1)^n. Otherwise, ciphertext is sampled from continuous Gaussian with covariance matrix r^2*G^(-1), and taken modulo 1.

Parameters
pkThe public key, i.e. G = (B^T)B
bPlaintext bit to be encrypted
Returns
The ciphertext
